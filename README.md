## Pokeapi Frontend

Frontend App (vue.js 2.x) para pesquisar e exibir informações básicas de todos os pokémons consumindo https://pokeapi.co/

#### Docker
 ```
$ docker-compose up --build
 ```

#### Local

- node v10
 ```
$ npm install -g @vue/cli
$ cd app
$ npm run serve
 ```

#### Tests
```
$ npm run test:unit
```

#### Build
```
$ npm run build
```
